1)Configure Git
git config --global user.name "w3schools-test"
git config --global user.email "test@w3schools.com"
2)Creating Git Folder
mkdir myproject--------mkdir makes a new directory.
cd myproject-----------cd changes the current working directory
3)Initialize Git
git init
4)ls will list the files in the directory. We can see that index.html is there.
5)git status
6)git add index.html=====Staged files are files that are ready to be committed to the repository you are working on.
7)git add --all
8)Git Commit git commit -m "First release of Hello World!"
9)Git Commit without Stage
git commit -a -m "Updcheckoutated index.html with a new line"
10)Git Commit Log
To view the history of commits for a repository, you can use the log command:
git log
11)Git help --all See All Possible Commands
To list all possible commands, use the help --all command:
$ git help --all
12)Git Branch
git branch hello-world-images
Now we created a new branch called "hello-world-images"
13)checkout---Moving us from the current branch, to
git checkout hello-world-images
Switched to branch 'hello-world-images'
14)Git Push to GitHub
*git commit -a -m "Updated index.html. Resized image"
*git push origin
